    • C'est quoi?

    Les FlexBox (Flexible Box) et Grid CSS (grilles), ce sont des propriétés CSS qui vont nous faciliter l'organisations d'élements dans notre site.
    Ces propriétés CSS vont nous permettre de créer des grilles ou des sortes de tableaux dans les quels on aura des éléments ou blocs, qu'on pourra transformer et afficher de façon très flexible, pour avoir le rendu que l'on veut.

    Vu que mes explication théoriques sont un peu vagues, un peu plus en bas j'ai mis des explications techniques avec plus de détail.

    Utilisation des FlexBox

    On peut utiliser les FlexBox dans tous les navigateurs récents (Chrome, Edge, Mozilla, Safari, Opera..).
    Pour les mettre en place, il nous faudra créer un div qui fera de "container" pour nos blocs, il faudra lui ajouter une classe dans la quelle on y mettra la propriéte display: flex;, qui nous permettra de définir notre div comme une flex-box.

    .flex-container { display: flex; }

    Il suffit d'ajouter des elements dans notre "container", pour que la propriété display: flex; commencent a les arranger.
    Pour changer l'organisation des elements à l'intérieur du "container" il suffit d'ajouter la propriété flex-direction: row; pour afficher nos éléments par rangée, ou flex-direction: column; pour les afficher par colonnes.

    Pour que les elements de notre container se mettent a la ligne quand il n'y a plus de place, on peut lui ajouter la propriété flex-flow: wrap;.

    Avec justify-content: flex-start | center | flex-end; on peut aligner les elements de notre flexbox au début de celui-ci, au centre ou a la fin.
    On peut aussi les aligner et les spacer avec la propriété justify-content: space-around | space-between; qui va faire en sorte de centrer et spacer les elements dans le flex.
    Il existe la possibilité de aligner les elements de notre flexbox verticalement, avec la propriété align-items: flex-start | center | stretch | flex-end;.

    Utilisation des Grid CSS

    On peut utiliser les Grid CSS dans tous les navigateurs modernes (Chrome, Edge, Mozilla, Safari, Opera..).

    Pour les mettre en place, il nous faudra créer un div qui fera de "container" pour nos blocs, il faudra lui ajouter une classe dans la quelle on y mettra la propriéte display: grid;, qui nous permettra de définir notre div comme une grid.

    .grid-container { display: grid; }

    On à des propriétés qui nous permettent de positionner nos éléments dans les colonnes et rangées que l'on veut.
    Par exemple, on peut définir l'emplacement de notre item dans les colonnes, avec grid-column: 2 / 4;, les numéros 2 et 4 représentent les lignes de notre tableau, lignes des colonnes et rangées.
    La même chose peut se faire pour changer le positionnement d'un element dans nos rangées, ça se fait avec grid-row: 1 / 3;.
    Ces deux propriétés peuvent se mélanger pour définir avec précision l'emplacement de chacun de nos élements.

    Grace a la propriété gap, on va pouvoir espacer les éléments de notre grid.
    Ça peut se faire de différentes façons, avec grid-gap: 50px; pour espacer les colonnes et rangées de notre grid, ou avec grid-row-gap: 50px; pour espacer les rangées et grid-column-gap: 50px; pour espacer nos colonnes.

    On peut limiter la quantité de colonnes et rangées de notre grid grace a grid-column-start: 1; et grid-column-end: 4; qui nous permet de définir la ligne de départ et de fin de notre grid.
    Ça peut se faire aussi pour les rangées de notre grid avec grid-row-start: 1; et grid-row-end: 6; pour limiter le nombre de rangées de notre grid.

    comparaison entre les FlexBox et les Grid CSS

    Ici on a un div avec une class grid, qui va afficher 3 colonnes en répetition, donc dans chaque rangée.

    .grid { display: grid; grid-template-columns: repeat(3, 1fr); }

    1 ~ grid-column: 1 / 4;
    2
    3 ~ grid-row: 2 / 4;
    4
    5


    Ici on a un div avec une class flex, qui va afficher 3 colonnes en répetition, donc dans chaque rangée.

    .flex { display: flex; flex-wrap: wrap; }

    1
    2
    3
    4
    5 ~ Les blocs peuvent occuper tout l'espace disponible
    (quand des colonens manquent) de façon automatique.
